Neologismos

Un generador de verbos usando una red neuronal LSTM.

Un valor alto produce resultados más osados, pero aumenta la posibilidad de producir combinaciones incorrectas.


                    

                    

                        

                    

                

                

                

                    

                        

                    

                

                

                    Realizado por @kokuma usando ml5.js para las cosas de Machine Learning y mini.css para ponerlo todo bonito. Red neuronal entrenada con training-charRNN, usando los lemarios de Ismael Olea e Ignacio Mario Morales. Código y explicaciones disponibles en GitHub.
